# BalanceX
Automated CLMM liquidity management platform on Solana.

## Structure
| Folder | Description |
|--------|-------------|
| `backend/` | NestJS REST API (TypeORM + PostgreSQL + Redis) |
| `frontend/` | React + Reown AppKit |
| `contracts/` | Solana Anchor smart contracts |

## Tech Stack
- **Blockchain**: Solana (Mainnet)
- **Protocol**: Raydium CLMM (Concentrated Liquidity)
- **Backend**: NestJS, TypeORM, PostgreSQL, Redis
- **Frontend**: React, Reown AppKit, Anchor SDK
